Introduction

gack is a Go port of the ack code searching utility. It will eventually support the same interface as the original ack.

Background

I started working on gack when I saw a port of ack to the C language. As I am currently working to learn Go, and the way for me to do that is to write as much Go as possible, I thought it would be an interesting project.

Status

gack currently supports the following flags:

        -f          list files only
        -g query    list files that match the query

Immediate TODO

  • support language flags to search or exclude certain languages

Bugs

  • gack uses the filepath.Walk function, which doesn't handle large directories well.
  • gack is currently slower than ack.
